Financial Math

Compound Annual Growth Rate (CAGR):

$$\mathrm{CAGR} = \left(\frac{\text{Ending Value}}{\text{Beginning Value}}\right)^{1/n} - 1$$

Net Present Value (NPV):

$$\mathrm{NPV} = \sum_{t=0}^{T} \frac{C_t}{(1+r)^t}$$

Return on Equity (ROE):

$$\mathrm{ROE} = \frac{\text{Net Income}}{\text{Average Shareholders' Equity}}$$

Walk through intuition: time value of money (NPV), compounding (CAGR), and profitability (ROE).

Reproducible KPIs (Code)

# Example: EBITDA margin and interest coverage
from dataclasses import dataclass

a = lambda x: round(x, 2)

@dataclass
class Financials:
    revenue: float
    ebitda: float
    ebit: float
    interest_expense: float

f = Financials(revenue=3200.0, ebitda=910.0, ebit=650.0, interest_expense=180.0)

ebitda_margin = f.ebitda / f.revenue  # 28.4%
interest_coverage = f.ebit / f.interest_expense  # 3.6x

print({
    "EBITDA_margin": a(ebitda_margin * 100),
    "Interest_coverage": a(interest_coverage)
})

-- Illustrative: revenue by segment (Q2)
SELECT segment,
       SUM(revenue) AS rev,
       SUM(expense) AS opex,
       SUM(revenue) - SUM(expense) AS op_income
FROM ledger
WHERE quarter = '2025Q2'
GROUP BY segment
ORDER BY rev DESC;
